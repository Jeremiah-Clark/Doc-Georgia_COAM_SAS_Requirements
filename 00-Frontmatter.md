# Description

This document is designed for experienced slot developers and producers who need to quickly familiarize themselves with Georgia COAM regulations and requirements. 
It provides comprehensive lists of mandatory SAS events, extended polls, and meters, along with the restrictions imposed on secondary SAS connections.
